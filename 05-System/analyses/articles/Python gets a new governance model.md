## 可仿写元素

- **技术新闻的倒金字塔结构**：先说结果，再展开背景和细节，最后展望未来，适合快速传递核心信息
- **多源引用编织法**：将讨论帖、邮件列表、官方文档、个人观点等多种来源有机整合，构建立体叙事
- **技术概念的渐进式解释**：先提出现象（投票争议），再解释原理（不同机制的优劣），最后给出解决方案（采用Condorcet）
- **关键人物的直接引语穿插**：在技术描述中插入参与者的原话，增加报道的真实感和人情味
- **小标题引导的信息分层**：使用"The winner"、"The steering council"等简洁小标题帮助读者快速定位信息
