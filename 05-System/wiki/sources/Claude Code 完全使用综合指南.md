# Claude Code 完全使用综合指南

## 核心论点
一篇面向中文开发者的 Claude Code 入门到进阶教程，强调 CLI 形态带来的原生 AI 能力（无阉割模型、无限工具调用、200K+ 上下文），并通过安装配置、核心命令、权限管理、思考模式、高级特性等章节系统介绍使用方法。

## 关键信息
- **定位**：基于命令行的智能编程助手，区别于传统 AI IDE。
- **核心卖点对比**：以表格形式对比普通 AI IDE 与 Claude Code 在 AI 能力、工具调用次数、上下文窗口、自主执行、系统状态感知五个维度的差异。
- **安装方式**：国内镜像站（ClaudeYY）一键安装脚本 vs 官方 npm 安装。
- **配置体系**：全局 `~/.claude.json` + 项目级 `settings.json` + 环境变量。
- **权限管理**：Interactive / Allowlist / Dangerous 三级权限，支持按工具前缀授权。
- **思考模式**：通过 `think` / `think hard` / `ultrathink` 等关键词控制推理深度。
- **持久化记忆**：`/init` 生成 `CLAUDE.md`，支持用户级与项目级记忆文件。
- **深度体验**：后半部分以个人使用视角补充，涵盖项目解读、编码辅助、知识补充、规范约束等场景。

## 可借鉴的写作结构
1. 痛点对比表开篇，快速建立产品差异认知。
2. 安装-配置-命令-安全-高级特性，按使用链路递进。
3. 末尾加入真实使用体验，软化教程的机械感。
