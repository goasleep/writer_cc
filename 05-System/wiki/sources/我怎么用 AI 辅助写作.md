## 核心论点

作者分享了用AI辅助写作的5步实践流程，其中第5步经历了重要演进：从"让AI模仿文风写作"转变为"自己口述+AI润色"。

## 5步写作流程

### 第一步：有所感
- 看书、翻文章，或与同事朋友聊天
- 只要有感触，第一时间记下来

### 第二步：和AI讨论
- 拿着感触和观点与AI讨论
- AI知识更丰富，思考更深
- 经常提供新名词、新想法，刺激灵感

### 第三步：提炼观点
- 基于与AI的讨论再思考
- 提炼出自己的观点
- 例："应该用AI辅助写作，要找到最适合自己的最佳实践"

### 第四步：梳理结构
- 哪怕只写三五百字，也要有结构
- 与AI一起列出全部信息和观点
- 探讨组合方式，几轮讨论后结构清晰

### 第五步：执行写作（演进版）

**旧方法**：
- 直接让AI按自己的文风写
- 喂给AI自己写过的内容
- 加上明确指令，让AI模仿风格输出

**新方法**（AI建议）：
1. 看一遍跟AI讨论的内容和结构
2. 丢开电脑，用语音把内容重新说一遍
3. 让AI整理
4. AI补上缺失的地方、修正错字、推敲细节、定稿

**AI的判断**：
- 推敲过程不可或缺，是写作中最关键的环节
- 不建议完全交出去

## 使用的工具

- **第一、五步**: Slax Note（笔记工具）
- **第二、三、四步**: Claude Code

## 核心洞察

1. **人机协作而非替代**: AI用于激发灵感、梳理结构，最终表达仍由人类完成
2. **流程优于工具**: 5步流程比具体工具更重要
3. **口述的价值**: 用语音说一遍比直接写更自然
4. **推敲不可替代**: AI可以整理、修正、推敲，但核心表达人类自己做
