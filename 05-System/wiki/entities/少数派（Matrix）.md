# 少数派（Matrix）

**类型**: 组织
**领域**: 科技媒体、写作社区
**网址**: https://sspai.com

## 简介

少数派（sspai.com）是一家专注于科技产品、效率工具和生活方式的中文媒体平台。Matrix是少数派的写作社区，主张分享真实的产品体验、有实用价值的经验与思考。社区会不定期挑选最优质的文章进行展示，文章代表作者个人观点，少数派仅对标题和排版略作修改。

## Matrix 写作社区

**主张**:
> 分享真实的产品体验，有实用价值的经验与思考

**内容特点**:
- 用户原创内容
- 真实体验和观点
- 实用价值导向
- 跨领域讨论（科技、教育、效率等）
