# AI 助理

AI 助理（AI Assistant）是指能够理解用户意图、自主执行任务、提供智能建议的 AI 系统。

## 产品形态

### 本地部署型

1. **[[OpenClaw]]（小龙虾）**
   - 特点：用户控制权优先
   - 定位："你来指挥的系统"

2. **[[Hermes Agent]]**
   - 特点：AI 自学习优先
   - 定位："自己会长大的助手"

### 云端服务型

- Claude Code
- GitHub Copilot
- 各种 AI 编程助手

## 核心能力

1. **工具调用**：操控电脑、执行命令、访问文件
2. **多平台集成**：接 Telegram、飞书、企业微信等
3. **定时任务**：可设定时任务自动执行
4. **记忆系统**：记住用户偏好和项目信息
5. **Skills 扩展**：通过插件扩展能力

## 产品哲学分歧

> "AI 助理的终极形态不是'更聪明'，而是'不需要你重复说第二遍'。"

两种路径：
- **控制派**：给用户最大的控制权，人来指挥，AI 来执行
- **自学派**：让 AI 自己学会怎么干活，人只需要说一次

## 安全考虑

- **模型级安全**：依赖大模型判断力（OpenClaw）
- **框架级安全**：授权审批、安全扫描、容器隔离（Hermes Agent）
