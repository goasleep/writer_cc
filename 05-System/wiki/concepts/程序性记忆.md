# 程序性记忆

程序性记忆（Procedural Memory）是指关于"如何做某事"的记忆，区别于陈述性记忆（关于"是什么"的记忆）。

## 在 AI Agent 中的应用

### [[Hermes Agent]] 的实现

Hermes Agent 将程序性记忆概念应用到 AI 助理中：

1. **自动提炼**：完成一个复杂任务（5 次以上工具调用）后，自动把方法论提炼成 Skill
2. **动态优化**：后续使用中发现更好路径会自动更新
3. **Token 优化**：三级渐进式加载（平时只加载技能名称和描述，约 3000 token，需要时才加载完整内容）

### 对比

与 [[OpenClaw]] 的对比：
- **OpenClaw**：人工编写的 Markdown Skills，装什么会什么
- **Hermes**：AI 自动生成和优化，"自己学会怎么干活"

## 认知科学背景

程序性记忆在人类认知中对应技能和习惯，如骑自行车、打字等，一旦形成就难以用语言描述，但能高效执行。Hermes Agent 借鉴了这一概念，让 AI 通过实践自动积累操作经验。
