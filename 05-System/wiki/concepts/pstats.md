# pstats

File format used by Python's built-in profilers to store profiling data. The format is compatible across `profile`, `cProfile`, and the new `profiling.sampling` (Tachyon) profiler.

## Usage

Tools like `profiling-explorer` can read pstats files to provide visualization and analysis of profiling data.
