# profiling-explorer

A web-based tool for exploring Python profiling data stored in pstats files. Created by Adam Johnson and sponsored by Rippling.

## Features

- Dark mode
- Sortable columns (calls, internal time, cumulative time)
- Search/filter by filename or function name
- Copy buttons for file locations
- Caller/callee navigation

## Usage

```bash
uvx profiling-explorer <name>.pstats
```
