# 前端性能优化

前端性能优化不仅关乎用户体验，也直接影响后端负载和系统可扩展性。在高并发场景下，每一个不必要的 API 调用都会被放大成系统性风险。

## 核心策略

- **缓存**：浏览器缓存、客户端状态缓存、CDN 边缘缓存
- **防抖与节流**：控制用户输入、滚动等高频触发场景的请求频率
- **请求合并**：批量请求、GraphQL、请求分组
- **异步与增量更新**：优先渲染已有数据，后台静默刷新
- **优雅降级**：缓存兜底、错误提示、指数退避重试
- **监控观测**：前端 RUM 工具跟踪真实用户环境的 API 性能

## 设计原则

> 前端开发者不仅是 API 的使用者，也可以是 API 设计的推动者。
