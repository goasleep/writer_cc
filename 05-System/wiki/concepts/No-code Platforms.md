# No-code Platforms

Platforms for building AI agents without writing code. Examples include n8n, Dify, Langflow, and Lindy.

## Benefits

- Building an agent takes 15-60 minutes
- Accessible to business users, not just engineers
- Good for understanding what agents do before writing code
