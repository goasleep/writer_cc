# 仿写工作流完整示例

> 以朱自清《荷塘月色》为例，演示完整仿写流程

---

## 示例背景

**参考文章**：朱自清《荷塘月色》（节选）

**仿写主题**：秋日黄昏的公园

**目标**：学习朱自清散文的细腻描写和抒情风格

---

## Step 1: 原文呈现

```
曲曲折折的荷塘上面，弥望的是田田的叶子。叶子出水很高，像亭亭的舞女的裙。
层层的叶子中间，零星地点缀着些白花，有袅娜地开着的，有羞涩地打着朵儿的；
正如一粒粒的明珠，又如碧天里的星星，又如刚出浴的美人。微风过处，送来缕缕
清香，仿佛远处高楼上渺茫的歌声似的。这时候叶子与花也有一丝的颤动，像闪电
般，霎时传过荷塘的那边去了。
```

---

## Step 2: 使用 Claude Code 进行分析

### 2.1 结构分析 Prompt

```
请对以下文章进行深度结构分析：

【文章内容】
曲曲折折的荷塘上面，弥望的是田田的叶子。叶子出水很高，像亭亭的舞女的裙。
层层的叶子中间，零星地点缀着些白花，有袅娜地开着的，有羞涩地打着朵儿的；
正如一粒粒的明珠，又如碧天里的星星，又如刚出浴的美人。微风过处，送来缕缕
清香，仿佛远处高楼上渺茫的歌声似的。这时候叶子与花也有一丝的颤动，像闪电
般，霎时传过荷塘的那边去了。

请按以下框架分析：
1. 整体架构
2. 逻辑流程
3. 节奏控制
4. 可借鉴的结构技巧
```

### 2.2 预期分析结果

**整体架构**：
- 类型：写景抒情散文
- 结构：空间顺序（整体→局部→细节）
- 段落功能：单一段落，由远及近、由静到动

**逻辑流程**：
- 起点：荷塘全景（叶子）
- 展开：叶子细节→花朵描写→香气描写→动态描写
- 层次：视觉（叶子→花）→嗅觉（香）→动态（颤动）

**节奏控制**：
- 开头：舒缓，叠词营造韵律
- 中段：细腻，层层递进
- 结尾：突然加快（"霎时"）

**可借鉴技巧**：
1. 叠词运用（曲曲折折、田田、亭亭、层层、缕缕）
2. 比喻连珠（三个"如"字排比）
3. 通感修辞（清香→歌声）
4. 动静结合（静态描写→动态收束）

## Step 4: 技巧提取

### 4.1 技巧提取表

| 技巧类别 | 具体技巧 | 原文示例 | 仿写要点 |
|----------|----------|----------|----------|
| 叠词运用 | 音韵美 | 曲曲折折、田田、亭亭 | 选择合适场景用叠词 |

## Step 5: 创建仿写模板

### 5.1 结构模板

```
【秋日黄昏公园仿写模板】

整体结构：
- 单一段落，约200字
- 空间顺序：全景→局部→细节
- 感官顺序：视觉→嗅觉→动态

内容要点：
1. 开头：公园全景（树木/草地），用叠词
2. 中段：聚焦细节（落叶/夕阳），用博喻
3. 过渡：嗅觉描写（桂花香），用通感
4. 结尾：动态收束（风吹落叶），动静结合

字数分配：
- 全景：50字
- 细节：100字
- 香气：30字
- 动态：20字
```

### 5.2 风格指南

```
【风格要求】

词汇：
- 使用书面语
- 多用叠词（如：层层叠叠、斑斑点点）
- 使用色彩词（金黄、橙红、暖黄）

句式：
- 长句为主
- 修饰语丰富
- 分号连接并列内容

修辞：
- 必须使用博喻（一个本体三个喻体）
- 使用通感（如：香味→音乐）
- 动静结合

语气：
- 温柔细腻
- 含蓄内敛
- 略带忧郁的诗意
```

## Step 6: 仿写创作

### 6.1 仿写 Prompt

```
请使用以下模板进行仿写创作：

【结构模板】
- 单一段落，约200字
- 空间顺序：全景→局部→细节
- 感官顺序：视觉→嗅觉→动态

【风格指南】
- 词汇：书面语、叠词、色彩词
- 句式：长句、修饰丰富
- 修辞：博喻、通感、动静结合
- 语气：细腻、含蓄、诗意

【仿写主题】
秋日黄昏的公园

【必须使用的技巧】
1. 叠词运用
2. 博喻修辞（一个本体三个喻体）
3. 通感修辞
4. 动静结合

请创作，并标注使用的技巧。
```

### 6.2 仿写作品示例

```
宽宽阔阔的公园中央，铺展的是层层叠叠的落叶。落叶堆积很厚，像柔柔的地毯覆
盖着大地。金黄的地毯中间，零星地散落着些枫叶片，有翩翩然旋转着的，有静静
然躺卧着的；正如一只只的蝴蝶，又如天边燃烧的晚霞，又如岁月写就的信笺。微
风过处，送来阵阵甜香，仿佛远处琴房里悠扬的琴声似的。这时候落叶与光也有一
丝的跃动，像波光般，霎时掠过草地的那边去了。

【技巧标注】
- 叠词：宽宽阔阔、层层叠叠、翩翩然、静静然
- 博喻：落叶如蝴蝶、如晚霞、如信笺
- 通感：甜香→琴声
- 动静结合：静态落叶→动态跃动
```

## Step 7: 质量评估

### 7.1 评估 Prompt

```
请评估以下仿写作品：

【原文】
曲曲折折的荷塘上面，弥望的是田田的叶子。叶子出水很高，像亭亭的舞女的裙。
层层的叶子中间，零星地点缀着些白花......

【仿写作品】
宽宽阔阔的公园中央，铺展的是层层叠叠的落叶。落叶堆积很厚，像柔柔的地毯覆
盖着大地。金黄的地毯中间，零星地散落着些枫叶片......

请从结构、风格、技巧、原创性、整体质量五个维度评分。
```

### 7.2 预期评估结果

| 维度 | 评分 | 说明 |
|------|------|------|
| 结构相似度 | 8/10 | 空间顺序一致，层次清晰 |
| 风格还原度 | 8/10 | 叠词、长句、细腻风格接近 |
| 技巧运用 | 9/10 | 博喻、通感、动静结合都使用 |
| 原创性 | 9/10 | 内容完全原创 |
| 整体质量 | 8/10 | 作为独立作品质量良好 |

**总分**：8×0.25 + 8×0.25 + 9×0.2 + 9×0.2 + 8×0.1 = **8.3分**

**等级**：良好

**反馈**：
- 优点：技巧运用到位，风格把握准确
- 不足：个别用词可更精炼
- 建议：多读原文，体会情感深度

---

## Step 8: 迭代改进

### 8.1 改进方向

根据评估反馈，可改进之处：

1. **情感深度**：原文有淡淡的忧愁，仿写可加强
2. **用词精炼**："宽宽阔阔"可改为更贴切的叠词
3. **意境营造**：可增加更多秋日特有的意象

### 8.2 改进版仿写

```
疏疏落落的梧桐树下，铺满的是层层叠叠的落叶。落叶积得深厚，像岁月编织的毯
子覆盖着往事。金黄的地毯中间，零星地散落着些枫叶片，有翩翩然旋舞着的，有
寂寂然沉睡着的；正如一只只疲倦的蝴蝶，又如天边褪色的记忆，又如时光写就的
诗行。晚风过处，送来缕缕幽香，仿佛远处琴房里断续的琴声似的。这时候落叶与
影也有一丝的颤动，像叹息般，霎时漫过草地的那边去了。

【改进点】
- "疏疏落落"比"宽宽阔阔"更有秋意
- "岁月编织的毯子覆盖着往事"增加情感深度
- "褪色的记忆"、"时光写就的诗行"更有意境
- "晚风"、"断续"、"叹息"增强忧郁氛围
```

## 学习要点

1. **分析是仿写的基础**：只有深入理解原文，才能有效模仿
2. **模板是创作的脚手架**：有了模板，仿写更有方向
3. **技巧要理解原理**：知道为什么用，才能用得恰当
4. **评估促进进步**：客观评估帮助发现问题
5. **迭代才能精进**：好文章是改出来的
