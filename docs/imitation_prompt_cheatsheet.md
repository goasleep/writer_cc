## 一、文章分析类

### A1. 快速分析（30秒版）
```
请快速分析这篇文章的结构、风格和技巧：

【文章】
{粘贴文章}

请用简洁的方式输出：
1. 一句话概括核心内容
2. 结构类型
3. 3个风格关键词
4. 2个最值得学习的技巧
```

### A2. 深度结构分析
```
请对以下文章进行深度结构分析：

【文章内容】
{粘贴文章内容}

请按以下框架分析：

1. **整体架构**
   - 文章类型与体裁
   - 采用的结构模式
   - 段落数量与功能分布
   - 字数分布比例

2. **逻辑流程**
   - 核心论点/主题
   - 论证/叙述展开顺序
   - 层次递进关系
   - 转折点位置

3. **节奏控制**
   - 行文快慢变化
   - 详略安排策略
   - 高潮/重点位置

4. **可借鉴的结构技巧**
```

### A3. 风格特征分析
```
请分析以下文章的风格特征：

【文章内容】
{粘贴文章内容}

请分析：
1. **语言特征** - 词汇偏好、句式特点、标点运用
2. **叙述特征** - 视角人称、时间处理、空间布局
3. **情感基调** - 整体色彩、变化曲线、态度倾向
4. **独特标识** - 标志性表达、个人化用词、句式习惯
5. **风格关键词** - 用3-5个词概括

请提供具体文本例证。
```

### A4. 写作技巧提取
```
请提取以下文章的写作技巧：

【文章内容】
{粘贴文章内容}

请提取：
1. **开头技巧** - 手法、效果、仿写要点
2. **过渡技巧** - 衔接方式、连接手段
3. **论证/叙述技巧** - 主要方法、例证策略
4. **修辞技巧** - 手法、频率、位置
5. **结尾技巧** - 收束方式、效果

请以表格输出：技巧名称 | 原文示例 | 效果分析 | 仿写要点
```

## 二、仿写创作类

### W1. 完整文章仿写
```
请仿写以下文章，主题改为【新主题】：

【原文】
{粘贴原文}

【仿写要求】
1. 保持原文结构
2. 模仿原文风格
3. 运用原文技巧
4. 内容完全原创
5. 字数相近

请先分析原文特点，再进行仿写。
```

### W2. 段落级仿写
```
请仿写以下段落：

【原文段落】
{粘贴段落}

【新内容】
{描述要写的内容}

要求：
- 保持句式结构
- 模仿语言节奏
- 使用类似修辞
- 内容原创
- 情感基调一致

请先分析原文特点，再仿写。
```

### W3. 风格仿写
```
请用【作者名】的风格写一段关于【主题】的文字：

【风格参考】
{粘贴风格样本}

【要求】
1. 完全模仿该风格
2. 内容原创
3. 字数控制在【字数】
```

### W4. 基于模板仿写
```
请使用以下模板进行仿写创作：

【结构模板】
- 第1段：引入，约100字
- 第2-4段：主体论证，每段150字
- 第5段：总结升华，约100字

【风格指南】
- 词汇：简洁有力，多用动词
- 句式：长短交替，善用短句
- 语气：客观冷静，略带批判

【技巧要求】
- 开头使用设问
- 中间使用对比论证
- 结尾使用金句升华

【仿写主题】
{输入主题}

请创作并标注使用的技巧。
```

## 三、风格迁移类

### S1. 单风格迁移
```
请将以下内容改写成【风格描述】的风格：

【原文】
{粘贴原文}

【目标风格特征】
- 词汇：___
- 句式：___
- 修辞：___
- 语气：___

请保持核心信息，完全转换风格。
```

### S2. 多风格对比
```
请用三种风格改写以下内容：

【原文】
{粘贴原文}

【风格1】：{描述}
【风格2】：{描述}
【风格3】：{描述}

请分别输出三个版本，并说明每种风格的特点。
```

### S3. 作者风格仿写
```
请用【作者名】的风格重写以下内容：

【原文】
{粘贴原文}

【该作者风格特征】
{粘贴风格分析}

要求：
1. 保持原文核心信息
2. 完全采用目标风格
3. 运用典型句式和修辞
4. 调整叙述视角和情感基调

请先说明迁移策略，再改写。
```

## 四、素材重组类

### M1. 多源素材整合
```
请基于以下素材创作一篇新文章：

【素材1】来自{文章A}
{素材内容}

【素材2】来自{文章B}
{素材内容}

【素材3】来自{文章C}
{素材内容}

【创作要求】
- 主题：{主题}
- 体裁：{体裁}
- 字数：{字数}
- 风格：{风格}

【重组策略】
- 结构：采用素材X的框架
- 观点：融合素材Y和Z
- 案例：使用素材X的类型
- 表达：参考素材Y的风格

请创作原创文章，并说明素材使用情况。
```

### M2. 结构嫁接
```
请用【文章A】的结构，写【文章B】的主题：

【结构参考】
- 开头：场景引入（约100字）
- 主体：三段并列论证（每段200字）
- 结尾：呼应升华（约100字）

【内容主题】
{主题描述}

【风格要求】
{风格描述}

请保持结构一致，内容原创。
```

## 五、质量评估类

### E1. 综合质量评估
```
请评估以下仿写作品：

【原文】
{粘贴原文}

【仿写作品】
{粘贴仿写作品}

请评估：

1. **结构相似度** (0-10分)
2. **风格还原度** (0-10分)
3. **技巧运用** (0-10分)
4. **原创性** (0-10分)
5. **整体质量** (0-10分)

总分 = 结构×0.25 + 风格×0.25 + 技巧×0.2 + 原创×0.2 + 质量×0.1

请给出：
- 具体评分
- 详细反馈（优点/不足）
- 改进建议
- 总体等级（优秀/良好/合格/待改进/不合格）
```

### E2. 对比分析
```
请对比原文和仿写：

【原文】
{粘贴原文}

【仿写】
{粘贴仿写}

请分析：
1. 相似之处
2. 差异之处
3. 仿写成功的地方
4. 需要改进的地方
5. 具体改进建议
```

## 六、进阶技巧类

### X1. 多文章融合
```
请融合以下文章的优点进行创作：

【文章A - 结构参考】
{粘贴文章A}
优点：结构清晰

【文章B - 风格参考】
{粘贴文章B}
优点：语言优美

【文章C - 技巧参考】
{粘贴文章C}
优点：论证有力

【创作主题】
{输入主题}

请创作一篇融合三者优点的文章。
```

### X2. 风格对比学习
```
请用五种不同风格写同一主题：

【主题】
{输入主题}

【风格】
1. 鲁迅式冷峻
2. 张爱玲式华丽
3. 汪曾祺式淡远
4. 余秋雨式厚重
5. 韩寒式犀利

请分别创作，并分析每种风格的特点。
```

### X3. 反向仿写
```
请分析我的文章，并用【目标风格】改写：

【我的文章】
{粘贴文章}

【目标风格】
{描述风格}

请先分析我文章的特点，然后用目标风格改写。
```

## 七、快速工作流

### 7步仿写流程

```
Step 1: 选择参考文章
    ↓
Step 2: 使用A2/A3/A4进行深度分析
    ↓
Step 3: 提取模板和规则
    ↓
Step 4: 使用W1/W2/W3进行仿写
    ↓
Step 5: 使用E1进行质量评估
    ↓
Step 6: 根据反馈迭代改进
    ↓
Step 7: 保存归档
```

### 常用组合

| 场景 | Prompt组合 |
|------|-----------|
| 学习一篇文章 | A2 → A3 → A4 |
| 仿写一篇文章 | A2 → W1 → E1 |
| 风格学习 | A3 → W3 → E1 |
| 技巧学习 | A4 → W2 → E1 |
| 多源创作 | M1 → E1 |
| 风格迁移 | S1 → E2 |

|------|------|------|

| {粘贴段落} | 粘贴单个段落 | - |

| {风格描述} | 风格特征描述 | "简洁有力、冷峻客观" |
