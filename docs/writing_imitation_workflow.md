## 一、参考文章分析框架

### 1.1 三维分析模型

```
┌─────────────────────────────────────────────────────────────┐
│                    参考文章三维分析模型                        │
├─────────────────┬─────────────────┬─────────────────────────┤
│    结构维度      │    风格维度      │      技巧维度           │
├─────────────────┼─────────────────┼─────────────────────────┤
│ • 整体架构       │ • 语言特征       │ • 开头技巧              │
│ • 段落组织       │ • 句式特点       │ • 过渡手法              │
│ • 逻辑流程       │ • 词汇偏好       │ • 修辞运用              │
│ • 节奏控制       │ • 情感基调       │ • 结尾方式              │
│ • 层次递进       │ • 视角选择       │ • 细节描写              │
└─────────────────┴─────────────────┴─────────────────────────┘
```

### 1.2 结构分析清单

#### A. 宏观结构（骨架）
| 分析项 | 具体问题 | 记录模板 |
|--------|----------|----------|
| 文章类型 | 这是什么类型的文章？ | 议论文/记叙文/说明文/散文/评论 |
| 整体框架 | 采用什么结构模式？ | 总分总/并列式/递进式/对比式/时间线 |
| 段落数量 | 共多少段？各段功能？ | 开头段__ 主体段__ 结尾段__ |
| 字数分布 | 各部分字数比例？ | 开头:__% 主体:__% 结尾:__% |
| 核心论点 | 中心思想是什么？ | 一句话概括：______ |

#### B. 中观结构（肌肉）
| 分析项 | 具体问题 | 记录模板 |
|--------|----------|----------|
| 段落功能 | 每段承担什么功能？ | 引入/论证/举例/过渡/总结 |
| 段落长度 | 段落长短变化规律？ | 长短交替/逐渐增长/保持稳定 |
| 内部结构 | 段落内部如何组织？ | 观点+例证/问题+解答/现象+分析 |
| 连接方式 | 段与段如何衔接？ | 关键词重复/逻辑连接词/意象延续 |

#### C. 微观结构（细胞）
| 分析项 | 具体问题 | 记录模板 |
|--------|----------|----------|
| 句子长度 | 长短句如何搭配？ | 长句__% 短句__% 平均__字/句 |
| 句子类型 | 使用哪些句式？ | 陈述/疑问/感叹/排比/对偶 |
| 开头方式 | 如何开篇？ | 设问/引用/场景/观点/故事 |
| 结尾方式 | 如何收束？ | 升华/呼应/留白/呼吁/总结 |

### 1.3 风格分析清单

#### A. 语言风格矩阵
```
                    简洁 ◄─────────────────► 繁复
                      │                      │
    理性 ◄────────────┼────────────────────┼────────► 感性
                      │                      │
    严肃 ◄────────────┼────────────────────┼────────► 活泼
                      │                      │
    直白 ◄────────────┼────────────────────┼────────► 含蓄
```

#### B. 风格特征提取表
| 维度 | 具体指标 | 分析要点 | 示例记录 |
|------|----------|----------|----------|
| 词汇 | 用词偏好 | 抽象/具体、书面/口语、专业/通俗 | 偏好四字成语、善用比喻 |
| 句式 | 句子结构 | 长短搭配、整散结合、标点运用 | 多用短句，善用破折号 |
| 修辞 | 修辞手法 | 比喻、排比、拟人、夸张、对比 | 比喻密度高，善用通感 |
| 语气 | 叙述口吻 | 客观/主观、热情/冷静、幽默/严肃 | 冷静客观中带有温情 |
| 节奏 | 行文节奏 | 快慢变化、张弛有度、抑扬顿挫 | 开头慢，中段快，结尾慢 |
| 视角 | 叙述视角 | 第一/二/三人称、全知/限知 | 第一人称限知视角 |

| 结尾技巧 | 金句升华 | 末段 | 余韵悠长 | 提炼核心 |

## 二、Claude Code 仿写工作流

### 2.1 完整工作流图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  1.文章收集  │───►│  2.深度分析  │───►│  3.元素提取  │───►│  4.仿写创作  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
       │                  │                  │                  │
       ▼                  ▼                  ▼                  ▼
  建立素材库          三维分析          创建模板           迭代优化
  分类标签           填写清单          构建框架           质量评估
```

### 2.2 详细步骤说明

#### Step 1: 文章收集与预处理

**操作内容：**
1. 将参考文章保存到写作库
2. 添加元数据标签（类型、主题、风格、作者）
3. 进行初步阅读，标记重点段落

**Claude Code 操作：**
```bash
# 创建文章分析目录结构
mkdir -p ~/writing_library/{articles,analysis,templates,output}

# 保存参考文章
cp reference_article.md ~/writing_library/articles/
```

---

#### Step 2: 深度分析（使用 Claude Code）

**Prompt 模板 2.1 - 文章结构分析：**
```
请对以下文章进行深度结构分析：

【文章内容】
{粘贴文章内容}

请按以下框架分析：

1. **整体架构**
   - 文章类型与体裁
   - 采用的结构模式（总分总/并列/递进等）
   - 段落数量与功能分布
   - 字数分布比例

2. **逻辑流程**
   - 核心论点/主题
   - 论证/叙述的展开顺序
   - 层次递进关系
   - 转折点位置分析

3. **节奏控制**
   - 行文快慢变化
   - 详略安排策略
   - 高潮/重点位置

4. **结构创新点**
   - 独特之处
   - 可借鉴的结构技巧

请以结构化格式输出分析结果。
```

**Prompt 模板 2.2 - 风格特征分析：**
```
请分析以下文章的风格特征：

【文章内容】
{粘贴文章内容}

请从以下维度分析：

1. **语言特征**
   - 词汇偏好（抽象/具体、书面/口语）
   - 句式特点（长短搭配、整散结合）
   - 标点运用特点

2. **叙述特征**
   - 叙述视角与人称
   - 时间处理方式
   - 空间布局策略

3. **情感基调**
   - 整体情感色彩
   - 情感变化曲线
   - 作者态度倾向

4. **独特标识**
   - 标志性表达
   - 个人化用词
   - 句式习惯

5. **风格关键词**
   - 用3-5个词概括风格

请提供具体的文本例证支持你的分析。
```

**Prompt 模板 2.3 - 写作技巧提取：**
```
请提取以下文章的写作技巧：

【文章内容】
{粘贴文章内容}

请按以下类别提取：

1. **开头技巧**
   - 使用的手法
   - 效果分析
   - 仿写要点

2. **过渡技巧**
   - 段间衔接方式
   - 逻辑连接手段
   - 仿写要点

3. **论证/叙述技巧**
   - 主要论证方法
   - 例证选择策略
   - 仿写要点

4. **修辞技巧**
   - 使用的修辞手法
   - 出现频率与位置
   - 仿写要点

5. **结尾技巧**
   - 收束方式
   - 效果分析
   - 仿写要点

请以表格形式输出，包含：技巧名称、原文示例、效果分析、仿写要点
```

#### Step 5: 质量评估与迭代

**Prompt 模板 5.1 - 仿写质量评估：**
```
请评估以下仿写作品的质量：

【原文】
{粘贴原文}

【仿写作品】
{粘贴仿写作品}

请从以下维度评估：

1. **结构相似度** (0-10分)
   - 整体架构是否一致
   - 段落组织是否相似
   - 逻辑流程是否匹配

2. **风格还原度** (0-10分)
   - 语言特征是否接近
   - 叙述口吻是否一致
   - 情感基调是否吻合

3. **技巧运用** (0-10分)
   - 技巧识别是否准确
   - 技巧运用是否恰当
   - 技巧效果是否达成

4. **原创性** (0-10分)
   - 内容是否原创
   - 是否有抄袭嫌疑
   - 是否有个人创新

5. **整体质量** (0-10分)
   - 作为独立作品的质量
   - 可读性和吸引力
   - 完成度和精致度

请给出具体评分和详细反馈，指出改进方向。
```

### 4.2 素材重组策略

#### A. 重组模式
```
┌─────────────────────────────────────────────────────────────┐
│                      素材重组模式                             │
├─────────────────┬───────────────────────────────────────────┤
│ 模式名称        │ 说明                                      │
├─────────────────┼───────────────────────────────────────────┤
│ 结构嫁接        │ A的结构 + B的内容 + C的风格               │
│ 观点融合        │ A的观点1 + B的观点2 → 新观点              │
│ 案例重组        │ A的案例框架 + B的案例细节                 │
│ 表达借用        │ 借用A的句式，填入B的内容                  │
│ 多源整合        │ 多篇文章的素材，按新主题重组              │
└─────────────────┴───────────────────────────────────────────┘
```

#### B. 重组流程
```
1. 确定主题 → 明确写作目标和核心观点
     ↓
2. 素材检索 → 从库中搜索相关素材
     ↓
3. 素材筛选 → 选择最相关的素材
     ↓
4. 结构规划 → 设计文章整体框架
     ↓
5. 素材填充 → 将素材放入对应位置
     ↓
6. 风格统一 → 调整语言风格保持一致
     ↓
7. 原创转化 → 确保内容原创性
```

## 五、仿写质量评估方法

### 5.1 多维度评估体系

```
┌─────────────────────────────────────────────────────────────────┐
│                    仿写质量评估体系                              │
├───────────────────┬─────────┬───────────────────────────────────┤
│ 评估维度          │ 权重    │ 评估要点                          │
```

## 八、进阶技巧

### 8.1 多文章融合仿写

**方法：** 选择2-3篇不同文章，各取所长
- A的结构 + B的风格 + C的技巧

**Prompt：**
```
请融合以下三篇文章的优点进行创作：

【文章A - 结构参考】
{粘贴文章A}
【优点】：结构清晰

【文章B - 风格参考】
{粘贴文章B}
【优点】：语言优美

【文章C - 技巧参考】
{粘贴文章C}
【优点】：论证有力

【创作主题】
{输入主题}

请创作一篇融合三者优点的文章。
```

### 8.2 反向仿写

**方法：** 分析自己文章，用目标风格改写

**Prompt：**
```
请分析我的文章，并用【目标风格】改写：

【我的文章】
{粘贴文章}

【目标风格】
{描述风格}

请先分析我文章的特点，然后用目标风格改写。
```

### 8.3 风格对比学习

**方法：** 同一主题，用不同风格写作，对比学习

**Prompt：**
```
请用五种不同风格写同一主题：

1. 鲁迅式冷峻

4. 余秋雨式厚重
5. 韩寒式犀利

请分别创作，并分析每种风格的特点。
```

## 九、常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 仿写像抄袭 | 过于依赖原文表达 | 先理解再表达，用自己的话重写 |
| 风格不统一 | 分析不够深入 | 使用风格分析清单逐项对照 |
| 结构松散 | 未遵循模板 | 先列大纲，再填充内容 |
| 技巧生硬 | 技巧理解不透 | 分析技巧原理，理解后再用 |
| 缺乏新意 | 只模仿不创新 | 在模仿基础上加入个人观点 |
| 评估不准 | 缺乏客观标准 | 使用多维度评估体系 |
