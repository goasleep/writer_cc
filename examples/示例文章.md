# 如何构建高效的知识管理系统

## 核心观点

1. **知识管理的核心是连接** - 不是简单的收集，而是建立知识之间的关联
2. **PARA方法** - 项目(Projects)、领域(Areas)、资源(Resources)、归档(Archive)四层结构
3. **Zettelkasten笔记法** - 原子化笔记 + 永久编号 + 索引卡片
4. **定期回顾** - 每周回顾Inbox，每月整理标签

## 详细笔记

### PARA方法详解

PARA是Tiago Forte提出的知识组织框架：

- **Projects（项目）**: 有明确目标和截止日期的任务
- **Areas（领域）**: 需要长期维护的责任范围
- **Resources（资源）**: 感兴趣但暂时不用的参考资料
- **Archive（归档）**: 已完成或暂停的内容

### Zettelkasten笔记法

1. ** fleeting notes（闪念笔记）**: 临时记录想法
2. **literature notes（文献笔记）**: 阅读时的摘录
3. **permanent notes（永久笔记）**: 用自己的话重述，原子化
4. **index notes（索引笔记）**: 主题地图，组织相关笔记

## 个人思考

这篇文章与[[第二大脑]]的理念非常契合。我认为最重要的是：

1. 不要追求完美系统，先开始用起来
2. 定期回顾比完美分类更重要
3. 双向链接是发现知识关联的关键
