## 📊 质量分布概览

```dataview
TABLE
  quality_tier as "等级",
  length(rows) as "数量",
  round(average(map(rows, (r) => r.score_content_depth)), 1) as "平均深度",
  round(average(map(rows, (r) => r.score_readability)), 1) as "平均可读"
FROM "05-System/analyses/articles"
GROUP BY quality_tier
SORT quality_tier DESC
```

## 🎯 精品推荐（组合筛选）

### 深度且可读（content_depth ≥ 85 AND readability ≥ 85）

```dataview
TABLE
  file.link as "📄 文章",
  score_content_depth as "📖 深度",
  score_readability as "📖 可读",
  score_originality as "💡 原创"
FROM "05-System/analyses/articles"
WHERE score_content_depth >= 85 AND score_readability >= 85
SORT score_content_depth DESC
```

### 全能型选手（五维度 ≥ 80）

```dataview
TABLE
  file.link as "📄 文章",
  score_content_depth as "深度",
  score_readability as "可读",
  score_originality as "原创",
  score_structure as "结构",
  score_technique as "技巧"
FROM "05-System/analyses/articles"
WHERE score_content_depth >= 80
  AND score_readability >= 80
  AND score_originality >= 80
  AND score_structure >= 80
  AND score_technique >= 80
SORT score_content_depth DESC
```

### 技术深度文（技术类 + content_depth ≥ 85）

```dataview
TABLE
  file.link as "📄 文章",
  score_content_depth as "📖 深度",
  score_technique as "🛠️ 技巧",
  key_techniques as "🔑 关键技巧"
FROM "05-System/analyses/articles"
WHERE (contains(article_type, "technical") OR contains(article_type, "技术"))
  AND score_content_depth >= 85
SORT score_content_depth DESC
```
